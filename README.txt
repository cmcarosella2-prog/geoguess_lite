# GeoGuess Lite

## Setup Instructions
1. Get a Google API Key:
   - Go to https://console.cloud.google.com/
   - Enable Street View Static API.
   - Create an API key and replace 'YOUR_API_KEY' in script.js.

2. Deploy:
   - Upload files to Netlify or Vercel.
   - Connect your domain.

3. Monetize:
   - Sign up for Google AdSense.
   - Add AdSense script in <head> of index.html.

## Features
- Easy Mode: Guess country.
- Hard Mode: Guess city.
- Dynamic Street View images.
- Score tracking.
